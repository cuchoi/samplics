Sample selection¶

samplics.sampling.selection¶

Sampling selection module

The module has one main class called SampleSelection which provides a number of random selection methodsand associated probability of selection. All the samping techniques implemented in this modules are discussed in the following reference book: Cochran, W.G. (1977) 1, Kish, L. (1965) 2, and Lohr, S.L. (2010) 3. Furthermore, Brewer, K.R.W. and Hanif, M. (1983) 4 provides comprehensive and detailed descriptions of these complex sampling algorithms.

1

Cochran, W.G. (1977), Sampling Techniques, 3rd edn., Jonh Wiley & Sons, Inc.

2

Kish, L. (1965), Survey Sampling, Jonh Wiley & Sons, Inc.

3

Lohr, S.L. (2010), Sampling: Design and Analysis, 2nd edn., Cengage Learning, Inc.

4

Brewer, K.R.W. and Hanif, M. (1983), Sampling With Unequal Probabilities, Springer-Verlag New York, Inc

class samplics.sampling.selection.SampleSelection(method: str, stratification: bool = False, with_replacement: bool = True)[source]¶

Bases: object

SampleSelection implements a number of sampling selection algorithms.

The implemented sampling algorithms are the simple random sampling (srs), the systematic selection (sys), five different algorithms of probability proportional to size (pps), and the generic selection.

Attributes of the class:
method (str): a string specifying the sampling algorithm. The available
sampling algorithms are: simple random sampling (srs), systematic selection (sys),
Brewer’s procedure (pps-brewer), Hanurav-Vijayan procedure (pps-hv),
Murphy’s procedure (pps-murphy), Rao-Sampford procedure (pps-rs), systematic
pps (pps-sys), and a generic selection from arbitrary probabilities using
numpy.random.choice() method.
stratification (bool): indicates if the sampling procedure is stratified or not.
with_replacement (bool): indicates whether the selection is with replacement.
Main functions:
inclusion_probs(): provides the inclusion probabilities.
joint_inclusion_probs(): provides the joint probalities of selection.
select(): indicates the selected sample.
